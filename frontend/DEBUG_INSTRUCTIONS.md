# Инструкции по отладке кнопки сортировки

## Проблема
Кнопка сортировки не реагирует на клики.

## Шаги отладки

### 1. Откройте консоль браузера
- Нажмите F12 или Ctrl+Shift+I
- Перейдите на вкладку "Console"

### 2. Откройте страницу с кнопкой сортировки
- Откройте `dist/index.html` в браузере
- Или используйте `debug-sort-button.html` для тестирования

### 3. Проверьте сообщения в консоли
Должны появиться следующие сообщения:
```
CarsLoader init() вызван
DOM уже загружен, инициализируем сразу
Loading cars data, page 1, sort direction: desc...
Инициализация кнопки сортировки...
Найденная кнопка: <a class="btn btn-sort" href="#">
Кнопка сортировки инициализирована успешно
```

### 4. Нажмите на кнопку сортировки
Должны появиться сообщения:
```
Клик по кнопке сортировки!
toggleSortDirection вызван!
Направление сортировки изменено на: asc
Перезагружаем данные...
Обновляем визуальное состояние...
updateSortButtonVisual вызван
Кнопка для обновления: <a class="btn btn-sort" href="#">
Установлено состояние ASC
```

## Возможные проблемы и решения

### Проблема 1: "Кнопка сортировки не найдена на странице"
**Решение:**
- Убедитесь, что на странице есть элемент с классом `.btn-sort`
- Проверьте, что скрипт `cars_loader.js` загружается после HTML

### Проблема 2: "CarsLoader init() вызван" не появляется
**Решение:**
- Убедитесь, что скрипт `cars_loader.js` подключен к странице
- Проверьте путь к файлу: `assets/js/cars_loader.js`

### Проблема 3: Кнопка найдена, но клик не работает
**Решение:**
- Проверьте, нет ли других обработчиков событий, которые могут блокировать клик
- Убедитесь, что кнопка не перекрыта другими элементами

### Проблема 4: API запросы не отправляются
**Решение:**
- Проверьте, что API сервер запущен на `http://localhost:8000`
- Убедитесь, что API поддерживает параметр `sort_direction`

## Тестовые файлы

### 1. `debug-sort-button.html`
Простая тестовая страница для проверки работы кнопки без API.

### 2. `test-cars-listing.html`
Полная интеграция с API для тестирования.

## Команды для проверки

### В консоли браузера:
```javascript
// Проверить, есть ли кнопка на странице
document.querySelector('.btn-sort')

// Проверить, инициализирован ли carsLoader
window.carsLoader

// Проверить текущее направление сортировки
window.carsLoader.sortDirection

// Вручную переключить направление
window.carsLoader.toggleSortDirection()
```

### Проверка API:
```bash
# Проверить, работает ли API
curl "http://localhost:8000/cars?page=1&limit=15&sort_direction=desc"
```

## Ожидаемое поведение

1. **При загрузке страницы:**
   - Кнопка должна быть найдена и инициализирована
   - В консоли должны появиться сообщения об инициализации

2. **При клике на кнопку:**
   - Направление сортировки должно измениться
   - Визуальное состояние кнопки должно обновиться
   - Должен отправиться новый API запрос
   - Данные должны перезагрузиться

3. **Визуальные изменения:**
   - Кнопка должна менять цвет (синий для asc, фиолетовый для desc)
   - Подсказка должна обновляться
   - SVG иконка может поворачиваться

## Если ничего не помогает

1. Проверьте, что все файлы загружены без ошибок
2. Убедитесь, что нет конфликтов с другими скриптами
3. Попробуйте использовать `debug-sort-button.html` для изолированного тестирования
4. Проверьте, что API сервер работает и возвращает данные
