Демонстрация работы кнопки сортировки

Кнопка сортировки

Нажмите на кнопку ниже, чтобы переключить направление сортировки:

Текущее направление сортировки: desc (по убыванию)

Как это работает:

  • Клик по кнопке переключает направление сортировки между 'asc' и 'desc'
  • Визуальные изменения показывают текущее состояние (цвет, подсказка)
  • API запрос отправляется с параметром sort_direction
  • Данные перезагружаются с новым направлением сортировки

API запросы:

Последний запрос: GET /cars?page=1&limit=15&sort_direction=desc
