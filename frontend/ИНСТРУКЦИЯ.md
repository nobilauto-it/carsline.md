# Инструкция по использованию Carento Cars API

## Что было сделано

1. ✅ **Перенесен контейнер с фильтрами** из `cars-list-1.html` в `index.html`
2. ✅ **Создан Python API** для получения данных машин из PostgreSQL базы данных
3. ✅ **Заменены хардкодные данные** на динамические из API
4. ✅ **Настроено отображение только доступных полей** из базы данных

## Как запустить

### 1. Запуск API сервера

```bash
cd api
pip install -r requirements.txt
python run.py
```

API будет доступен по адресу: http://localhost:8000

### 2. Открытие сайта

Откройте файл `dist/index.html` в браузере. JavaScript автоматически загрузит данные машин из API.

## Структура данных

API возвращает только те поля, которые есть в базе данных:

**Доступные поля:**
- `marca.name` - Марка машины
- `model.name` - Модель машины  
- `price.base_value` - Цена
- `price.currency` - Валюта
- `fuel` - Тип топлива
- `transmission` - Тип трансмиссии
- `category` - Категория машины
- `img.url` - URL изображения

**Скрытые поля (их нет в базе):**
- `miles` - Пробег
- `seats` - Количество мест
- `location` - Местоположение

## Файлы

- `api/main.py` - Основной файл API
- `api/run.py` - Скрипт для запуска API
- `api/requirements.txt` - Зависимости Python
- `js/cars_loader.js` - JavaScript для загрузки данных
- `dist/index.html` - Главная страница с динамическими данными

## API Endpoints

- `GET /` - Информация об API
- `GET /cars` - Список всех машин
- `GET /cars/{id}` - Конкретная машина по ID
- `GET /docs` - Документация API (Swagger UI)
