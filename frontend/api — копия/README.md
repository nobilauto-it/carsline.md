# Carento Cars API

API для получения данных о машинах из базы данных PostgreSQL.

## Установка и запуск

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Запустите API сервер:
```bash
python main.py
```

API будет доступен по адресу: http://localhost:8000

## Эндпоинты

- `GET /` - Информация об API
- `GET /cars` - Получить список всех машин
- `GET /cars/{car_id}` - Получить конкретную машину по ID

## Структура данных

API возвращает данные машин со следующими полями:
- `id` - ID машины
- `created_date` - Дата создания
- `fuel` - Тип топлива
- `transmission` - Тип трансмиссии
- `category` - Категория машины
- `marca` - Марка машины
- `model` - Модель машины
- `price` - Цена
- `img` - Изображение машины
- `cover_img` - Обложка машины

## Интеграция с фронтендом

JavaScript загрузчик (`js/cars_loader.js`) автоматически:
1. Загружает данные с API
2. Отображает только доступные поля из базы данных
3. Скрывает поля, которых нет в базе (например, miles, seats)
4. Обновляет счетчик найденных машин
