# Документация по функционалу сортировки и фильтрации

## Обзор

Функционал сортировки и фильтрации автомобилей теперь полностью интегрирован с API и работает в реальном времени. Все изменения применяются немедленно и загружают новые данные с сервера.

## Компоненты

### 1. Кнопка сортировки (иконка с двумя стрелками)
- **Функция**: Переключение направления сортировки (по возрастанию/убыванию)
- **CSS класс**: `.btn-sort`
- **Поведение**: При клике меняет `sortDirection` между `'asc'` и `'desc'`

### 2. Dropdown "Show" (количество элементов)
- **ID**: `#dropdownSort2`
- **Варианты**: 20, 30, 50 элементов на странице
- **Поведение**: При выборе нового значения сбрасывает на первую страницу

### 3. Dropdown "Sort by" (критерий сортировки)
- **ID**: `#dropdownSort`
- **Варианты**:
  - Most Viewed (most_viewed)
  - Recently search (recently_searched)
  - Most popular (most_popular)
  - Top rated (top_rated)
- **Поведение**: При выборе нового критерия сбрасывает на первую страницу

## API Параметры

Все запросы к API теперь включают следующие параметры:

```javascript
{
    page: 1,                    // Номер страницы
    limit: 20,                  // Количество элементов на странице
    sort: 'most_viewed',        // Критерий сортировки
    direction: 'desc'           // Направление сортировки (asc/desc)
}
```

## Класс CarsLoader

### Новые свойства:
- `currentSort`: текущий критерий сортировки
- `currentLimit`: текущее количество элементов на странице
- `sortDirection`: направление сортировки ('asc' или 'desc')

### Новые методы:

#### `initEventHandlers()`
Инициализирует все обработчики событий для кнопок и dropdown'ов.

#### `toggleSortDirection()`
Переключает направление сортировки между 'asc' и 'desc'.

#### `setLimit(limit)`
Устанавливает новое количество элементов на странице.

#### `setSort(sortValue)`
Устанавливает новый критерий сортировки.

#### `getSortValue(sortText)`
Преобразует текст из dropdown'а в значение для API.

#### `updateDropdownText(dropdown, text)`
Обновляет текст в dropdown кнопке.

#### `setActiveItem(clickedItem)`
Устанавливает активный класс для выбранного элемента dropdown'а.

## Использование

### 1. Подключение к существующей странице

Убедитесь, что на странице есть следующие элементы:

```html
<!-- Кнопка сортировки -->
<a class="btn btn-sort" href="#" title="Переключить направление сортировки">
    <!-- SVG иконка -->
</a>

<!-- Dropdown Show -->
<button class="btn dropdown-toggle" id="dropdownSort2" type="button" data-bs-toggle="dropdown">
    <span>20</span>
</button>

<!-- Dropdown Sort by -->
<button class="btn dropdown-toggle" id="dropdownSort" type="button" data-bs-toggle="dropdown">
    <span>Most Viewed</span>
</button>

<!-- Контейнер для автомобилей -->
<div class="box-grid-tours">
    <div class="row" id="carsGrid">
        <!-- Автомобили будут загружены сюда -->
    </div>
</div>
```

### 2. Подключение скрипта

```html
<script src="dist/assets/js/cars_loader.js"></script>
```

### 3. Инициализация

Скрипт автоматически инициализируется при загрузке страницы. Глобальный объект `carsLoader` доступен для ручного управления:

```javascript
// Переключить направление сортировки
carsLoader.toggleSortDirection();

// Установить количество элементов
carsLoader.setLimit(30);

// Установить критерий сортировки
carsLoader.setSort('most_popular');

// Загрузить конкретную страницу
carsLoader.loadCars(2);
```

## Требования к API

API должен поддерживать следующие параметры запроса:

- `page`: номер страницы (начиная с 1)
- `limit`: количество элементов на странице
- `sort`: критерий сортировки
- `direction`: направление сортировки ('asc' или 'desc')

Пример запроса:
```
GET /cars?page=1&limit=20&sort=most_viewed&direction=desc
```

## Обработка ошибок

При ошибках загрузки данных отображается сообщение об ошибке с кнопкой "Попробовать снова", которая перезагружает страницу.
