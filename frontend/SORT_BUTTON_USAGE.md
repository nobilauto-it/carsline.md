# Использование кнопки сортировки

## Обзор

Кнопка сортировки (`.btn-sort`) теперь полностью функциональна и интегрирована с API. Она позволяет переключать направление сортировки данных автомобилей между возрастающим и убывающим порядком.

## Как это работает

### 1. HTML структура
```html
<a class="btn btn-sort" href="#" title="Переключить направление сортировки">
    <svg width="18" height="18" viewbox="0 0 18 18" fill="none" xmlns="http://www.w3.org/2000/svg">
        <path d="M8.25 6L5.25 3M5.25 3L2.25 6M5.25 3L5.25 15" stroke="currentColor" stroke-width="1.5" stroke-linecap="round" stroke-linejoin="round"></path>
        <path d="M9.75 12L12.75 15M12.75 15L15.75 12M12.75 15L12.75 3" stroke="currentColor" stroke-width="1.5" stroke-linecap="round" stroke-linejoin="round"></path>
    </svg>
</a>
```

### 2. JavaScript функционал

В `cars_loader.js` добавлены следующие методы:

#### `initSortButton()`
- Инициализирует обработчик событий для кнопки сортировки
- Вызывается автоматически при загрузке страницы

#### `toggleSortDirection()`
- Переключает `sortDirection` между 'asc' и 'desc'
- Перезагружает данные с новым направлением сортировки
- Обновляет визуальное состояние кнопки

#### `updateSortButtonVisual()`
- Обновляет CSS классы кнопки для визуального отображения состояния
- Изменяет подсказку (title) кнопки

### 3. API интеграция

При каждом клике на кнопку отправляется запрос к API с параметром `sort_direction`:

```
GET /cars?page=1&limit=15&sort_direction=desc
GET /cars?page=1&limit=15&sort_direction=asc
```

### 4. Визуальные состояния

#### По умолчанию (desc - убывание)
- Класс: `.btn-sort.sort-desc`
- Цвет: фиолетовый
- Подсказка: "Сортировка по убыванию (нажмите для возрастания)"

#### После клика (asc - возрастание)
- Класс: `.btn-sort.sort-asc`
- Цвет: синий
- Подсказка: "Сортировка по возрастанию (нажмите для убывания)"
- SVG поворачивается на 180 градусов

## Требования к API

API должен поддерживать параметр `sort_direction`:

- `sort_direction=desc` - сортировка по убыванию
- `sort_direction=asc` - сортировка по возрастанию

## Отладка

В консоли браузера будут выводиться сообщения:
- "Кнопка сортировки инициализирована" - при успешной инициализации
- "Направление сортировки изменено на: [asc/desc]" - при переключении
- "Кнопка сортировки не найдена на странице" - если кнопка отсутствует

## Совместимость

- Не требует изменений в HTML структуре
- Совместимо с Bootstrap 5
- Поддерживает все современные браузеры
